# Personal-Blog
create by nodejs
<h2>这是我用nodeJS写的一个单页面路由跳转博客网站</h2>
<p>，能实现的功能有登录注册 管理员权限 游客访客查看 分类添加删除 分类下内容的添加删除、博客文章的评论。html部分用swig写的， </p>
运行
<pre>
<code>
npm install
node app.js
</code>
</pre>
在localhost:8081打开
